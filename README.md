# Empathy LLM Prompts

A curated collection of advanced prompts designed to build empathy skills through structured practice and reflection with Large Language Models.

**Author:** Kat Kemner  
**License:** MIT (see LICENSE file)

## Overview

This repository contains a comprehensive library of empathy-building prompts designed for use with AI language models like ChatGPT, Claude, or similar LLMs. Each prompt guides you through structured exercises to develop different dimensions of empathy.

**Intended Audience:** 
- Individuals seeking personal growth in emotional intelligence
- Researchers studying empathy development
- Practitioners in counseling, coaching, or education
- Anyone interested in advanced prompt engineering for self-development

## How to Use These Prompts

1. **Choose a prompt** from the library based on the dimension you want to develop
2. **Copy the entire prompt** into your preferred LLM (ChatGPT, Claude, etc.)
3. **Follow the guided exercise** - most prompts are designed to go step-by-step with you
4. **Practice regularly** - empathy skills develop through consistent practice

### Tips for Best Results

- Set aside dedicated time for each exercise (15-45 minutes)
- Be honest and vulnerable in your responses
- Allow the AI to guide you through each step before moving forward
- Revisit prompts as you grow - you'll notice different insights over time
- Journal your insights after completing exercises
